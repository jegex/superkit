# Code of Conduct

Superkit adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) code of conduct.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Enforcement

Instances of abusive behavior may be reported by contacting the project maintainer.
